Overview

LWQ is an IND-CCA2 secure Key Encapsulation Mechanism (KEM) designed for the post-quantum era. Its security relies on the hardness of advanced lattice problems, remaining secure even against large-scale quantum computers. LWQ aims to provide an optimal balance between security, performance, and key sizes.

Design

The primary design goals of LWQ are simplicity, efficiency, and flexibility. It is designed with both software and hardware implementations in mind:

  • Optimized Error Distributions: Avoids complex sampling algorithms, making implementations easier to secure against side-channel attacks.
  • Power-of-two Moduli: Removes the need for expensive modular reductions, replacing them with fast bitwise operations. This significantly simplifies hardware design.

Security

LWQ provides robust post-quantum security levels. We provide multiple parameter sets targeting different NIST security categories (equivalent to AES-128, AES-192, and AES-256).
