Software

The software package for the LWQ protocol includes reference implementations designed for clarity and security, as well as highly optimized implementations for various architectures.

Available Packages

  • Reference Implementation (C): Pure C code prioritizing readability and constant-time execution to prevent timing attacks. Suitable for integration into general-purpose cryptographic libraries.
  • Optimized Implementation (AVX2): Specifically tuned for modern Intel and AMD processors utilizing Advanced Vector Extensions for maximum throughput.
